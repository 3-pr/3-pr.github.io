سياسة الخصوصية لإضافات المتصفح

إضافة LetterStremio

تنطبق التفاصيل التالية خصيصاً على إضافة LetterStremio:

1. البيانات الوظيفية (محلياً)

الإضافة لا تقوم بجمع أي بيانات شخصية للمطور. البيانات التالية تُعالج برمجياً لضمان عمل الإضافة فقط:

• بيانات الاعتماد: عنوان البريد الإلكتروني ومفتاح المصادقة الخاص بحساب Stremio (تُحفظ محلياً فقط عند تسجيل الدخول).

• بيانات المحتوى: معلومات الفيلم (العنوان، المعرف) من صفحة Letterboxd لتحديد المحتوى المطلوب إضافته.

2. معالجة البيانات

• استخدام مفتاح المصادقة للتواصل مع واجهة برمجة تطبيقات Stremio نيابة عنك.

• معالجة بيانات الفيلم لتنسيقها وإرسالها إلى مكتبتك في Stremio عند النقر على زر "إضافة إلى المكتبة".

3. تخزين البيانات

• يتم تخزين عنوان البريد الإلكتروني ومفتاح المصادقة محلياً فقط على جهازك (chrome.storage.local). لا نقوم بتخزين أي من بياناتك على خوادم خاصة بنا.

4. مشاركة البيانات

• تتم مشاركة البيانات مع واجهة برمجة التطبيقات الرسمية لـ Stremio (api.strem.io) فقط عند طلبك إضافة محتوى إلى مكتبتك.

إضافة LetterNuvio

تنطبق التفاصيل التالية خصيصاً على إضافة LetterNuvio، وهي تهدف لتوضيح كيفية تعاملنا مع بياناتك بكل شفافية لضمان الامتثال لسياسات الخصوصية:

1. جمع البيانات واستخدامها

لتمكين مزامنة المحتوى مع حساب Nuvio الخاص بك، نجمع البيانات التالية:

• بيانات الاعتماد: رمز الوصول (JWT) ومعرف الملف الشخصي (Profile ID). تُستخدم هذه البيانات لمصادقة عمليات الإضافة والمزامنة مع حسابك.

• بيانات المحتوى: (العنوان، سنة الإصدار، رابط البوستر، ومعرف IMDb). تُجمع هذه المعلومات من صفحة الفيلم على Letterboxd لتخزينها في مكتبتك.

• البيانات التقنية: يتم إرسال استعلامات نصية (عنوان الفيلم) إلى محرك بحث Cinemeta لإيجاد المعرفات الصحيحة في حال عدم توفرها في الصفحة.

2. مشاركة البيانات (أطراف ثالثة)

نحن لا نبيع بياناتك أبداً. نشارك البيانات فقط مع الخدمات التقنية اللازمة لعمل الإضافة:

• Nuvio Backend (Supabase): يتم إرسال كافة بيانات المكتبة والمصادقة إلى خادم Nuvio الرسمي (dpyhjjcoabcglfmgecug.supabase.co).

• Cinemeta API (v3-cinemeta.strem.io): للبحث عن معرفات الأفلام الناقصة. لا يتم إرسال أي بيانات شخصية هنا.

3. تخزين البيانات والاحتفاظ بها

• التخزين المحلي: يتم تخزين رموز المصادقة محلياً في المتصفح (chrome.storage.local) لسهولة الوصول.

• التخزين السحابي: بيانات المكتبة تُخزن سحابياً في قاعدة بيانات Nuvio المرتبطة بحسابك لضمان توفرها لك في أي وقت ومن أي مكان.

4. حقوق المستخدم وحذف البيانات

لديك الحق الكامل في الوصول إلى بياناتك أو طلب حذفها:

• يمكنك حذف بيانات الاعتماد المخزنة محلياً ببساطة عن طريق تسجيل الخروج من الإضافة أو حذفها من المتصفح.

• لحذف البيانات المخزنة في خوادم Nuvio، يجب عليك استخدام واجهة تطبيق Nuvio الرسمية.

إضافات أخرى

Browser Extensions Privacy Policy

LetterStremio Extension

The following details apply specifically to the LetterStremio extension:

1. Functional Data Usage (Local)

The developer does NOT collect or store any of your data. The following data is processed locally to ensure the extension functions:

• Credentials: Your Stremio email and auth key (stored only locally in your browser).

• Content Data: Movie metadata (title, IMDb ID) used to identify content on the page.

2. Data Processing

• Using the authentication key to communicate with Stremio's API on your behalf.

• Formatting and sending film metadata to your Stremio library when you click the "Add to Library" button.

3. Data Storage

• Your email and activation key are stored only locally on your device (chrome.storage.local). We never store any of your data on our own servers.

4. Data Sharing

• Your data is shared with Stremio's official API (api.strem.io) only when you request to add content to your library.

LetterNuvio Extension

The following details apply specifically to the LetterNuvio extension. This section outlines our data practices to ensure full compliance with privacy policies:

1. Data Collection and Usage

To enable synchronization with your Nuvio account, we collect the following data:

• Credentials: Your Access Token (JWT) and Profile ID. These are required to authenticate and sync requests with your account.

• Content Metadata: Movie/Series title, release year, poster URL, and IMDb ID. This information is extracted from the Letterboxd page to be saved to your library.

• Technical Data: Search queries (film titles) are sent to the Cinemeta search engine to resolve correct IDs if they are not available on the page.

2. Data Sharing (Third Parties)

We never sell your data. We only share it with the following essential services:

• Nuvio Backend (Supabase): All library and authentication data is sent to Nuvio's official backend (dpyhjjcoabcglfmgecug.supabase.co).

• Cinemeta API (v3-cinemeta.strem.io): Used for resolving movie IDs. No personal information is shared with this service.

3. Data Storage and Retention

• Local Storage: Authentication tokens are stored locally in your browser (chrome.storage.local) for seamless access.

• Cloud Storage: Library data is stored in the Nuvio cloud database associated with your account to ensure it remains accessible to you at any time.

4. User Rights and Data Deletion

You have the right to access or delete your data at any time:

• You can delete locally stored data by logging out or removing the extension.

• To delete cloud-stored data, please use the official Nuvio application interface.
